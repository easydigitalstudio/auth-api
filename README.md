# Base Microservice
## Install
```
git clone https://github.com/easydigitalstudio/base-microservice
cd base-microservice
npm i
```
## Config
```
cp .env.default .env
```
## Start
:exclamation: Run mongo before running the backend
```
npm start
```
Edit the `.env` file
## Test
Access [http://localhost:3001/](http://localhost:3001/)
